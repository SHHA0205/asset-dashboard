# 자산 통합 대시보드

10개 증권 계좌(국내 KRX + 해외 US)의 보유 종목과 평가 현황을 한 화면에서 조회하는 개인용 웹 대시보드입니다.

## 주요 기능

- **통합 대시보드**: 10개 계좌를 그리드로 동시 조회 (국내=블루, 해외=그린 구분)
- **시세 자동 갱신**: Yahoo Finance API 연동 (15~20분 지연 시세)
- **환율 통합**: USD/KRW 실시간 환율로 해외 자산 원화 환산
- **종목 검색**: 티커/종목코드/한글명 자동완성 (2글자 이상, 0.3초 디바운스)
- **로컬 저장**: IndexedDB에 계좌·종목 데이터 영속화
- **계좌 관리**: 계좌 추가/이름변경/삭제, 국내·해외 그룹 지정

## 실행 방법

```bash
cd asset-dashboard
npm install
npm run dev
```

- 프론트엔드: http://localhost:5173
- API 서버: http://localhost:3001

## 기술 스택

| 계층 | 기술 |
|------|------|
| 프론트엔드 | React + TypeScript + Vite |
| 백엔드 | Express (API 프록시) |
| 저장소 | IndexedDB (idb) |
| 시세 | Yahoo Finance 비공식 API |
| 환율 | Yahoo Finance / exchangerate.host |

## 프로젝트 구조

```
asset-dashboard/
├── server/           # Express API (시세/환율/검색 프록시)
├── src/
│   ├── api/          # API 클라이언트
│   ├── components/   # UI 컴포넌트
│   ├── store/        # IndexedDB + React Context
│   ├── types/        # TypeScript 타입
│   └── utils/        # 계산/포맷 유틸
```

## 사용 안내

1. 최초 실행 시 10개 샘플 계좌와 데모 종목이 자동 생성됩니다.
2. **⚙ 계좌 설정**에서 계좌를 추가·수정·삭제할 수 있습니다.
3. 각 계좌 카드의 **+ 종목** 버튼으로 보유 종목을 추가합니다.
4. 계좌 카드를 클릭하면 상세 뷰에서 종목 삭제, 현금 잔고 수정이 가능합니다.
5. 상단 **시세 새로고침** 버튼으로 수동 갱신 (5분마다 자동 갱신).

## API 연동 확인

앱 실행 후 상단 요약바에서 다음을 확인하세요.

- **API 연결됨** (녹색) → 시세·환율이 실시간으로 조회되고 있음
- **USD/KRW** 에 실제 환율 숫자 표시 (예: 1,518.88)
- 종목 현재가가 매입가와 다른 실제 시세로 표시

API 서버 단독 테스트:

```bash
node test-api.mjs
```

`✓ API OK` 가 출력되면 인터넷 연동이 정상입니다.

## 주의사항

- Yahoo Finance API는 비공식 엔드포인트로, 정책 변경 시 동작이 중단될 수 있습니다.
- 시세는 15~20분 지연 데이터입니다.
- 개인 전용 로컬 사용을 전제로 하며, 로그인/인증 기능은 포함하지 않습니다.
- 백신/회사 프록시 환경에서는 Node.js SSL 인증서 검증을 우회하도록 설정되어 있습니다 (로컬 전용).
